Header introduce contenido de navegación y es como un encabezado. Cada sección puede tener su propio "header". "Body" es una sección, una obligatoria llamada siempre body por defecto.

Intento explicar las diferentes etiquetas aunque en la web no tenga sentido. Por ejemplo, "main" no es una sección, es una forma de estructurar tus páginas donde se incluyen todas las secciones, artículos, incluso aside (si los queremos diferente en las diferentes páginas, sino, podemos dejarlo fuera de main para que se repita en todas las demás páginas) Comúnmente se encontrará entre el header y el footer.

Como inicio de sección podemos añadir un encabezado

"Section" sirve para agrupar cierto grupo de información. Ejemplos sería si hiciéramos un libro de recetas, las agruparíamos por arroces, pasta, y postres italianos
