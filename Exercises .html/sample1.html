Web básica

Pasos iniciales para la creación de la estructura de una web básica

  • Añade el "doctype html"
  • Añade su idioma con "html lang="es" (es importante por accesibilidad)
  • Debes incluirle la normativa meta charset="UFT-8"
  • Seguidamente abrimos la cabeza, "head", donde introduciremos un título "title", éste es el que aparecerá en la ventana del navegador
  • Ahora comenzaremos con el cuerpo, "body", y añadiremos "h1" como nuestro título, presidiendo nuestro cuerpo (es preferible que solo haya un "h1")
  • Comienza con un párrafo "p"

Puedes añadirle listados, imágenes o enlaces como veremos más adelante.

Ejemplos de etiquetas básicas, atributos universales y atributos específicos

  • Etiquetas básicas
    • `h1`, `h2`, `h3`, `h4`, `h5`, `h6` - headings (encabezados)
    • `p` - paragraph (párrafo)
    • `a` - anchor (enlace)
    • `div`, `span` - divisiones genéricas
    • `ul`, `ol`, `li` - listas
  • Atributos universales
    • `lang` - define el idioma
    • `id` - define el identificador
    • `class` - define la clase
    • `style` - define el estilo
    • `title` - define el título
  • Atributos específicos
    • `src`
    • `href`
    • `alt`
    • `width`
    • `height`
